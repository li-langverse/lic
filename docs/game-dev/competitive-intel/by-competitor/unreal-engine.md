# Unreal Engine — competitive notes

## Product intros (what they sell)

- **UE5 keynote:** Nanite, Lumen, World Partition, virtual production — **renderer + open world**, not Sequencer-first.
- **Sequencer docs:** Non-linear editor, camera cuts, MRQ for **video export**.

## UI/UX to steal

- Bottom **Sequencer** panel; toolbar (Create Camera, Render, Auto Key).
- **World** context in toolbar; playback transport.
- **Movie Render Queue** presets.
