# li-studio

World Studio shell — composes `import ui` chrome hooks with `import gui` paint IR (`studio_wire_gui_hd`). Full render/world compose is wave-d-28.

## Build

```bash
lic build src/lib.li -o li-studio
```

From the monorepo root, ensure `lic` is built: `./scripts/build.sh`.
