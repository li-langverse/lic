# Roblox Studio — competitive notes

**Media:** RB-* in [media-catalog.md](../media-catalog.md)

## Product intros

- Docs-first; **creator** focus (not AAA film).
- **Assistant** in mezzanine — agent competitor.

## UI/UX to steal

- **UI tab** + **Style Editor** for creator screen UI.
- Toolbar tabs: Home / Model / Avatar / **UI** / Script.
- Explorer + Properties; Toolbox for assets.
- Docking position selector (excellent UX pattern).
- Pin/collapse panels for small laptops.
