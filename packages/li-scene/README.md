# li-std-scene

Entity IDs, `Transform3` (`math.Vec3` + `math.Quat` + scale), parent/child hierarchy (`SceneNode.parent`), and `transform_compose` / `scene_world_transform` for physics ↔ renderer sync.

`workload_class=stub` until full graph storage and picks land (wave D).

See [SIMULATION_UI_READINESS.md](../../docs/physics/SIMULATION_UI_READINESS.md).

```bash
lic build src/lib.li -o li-std-scene
```
